# 基于Vue.js和Element-ui的商城后台管理系统

> ## 1.项目运行：

- cd 到 vue_shop目录下运行:

```
npm install 或者 cnpm install
```
- 运行：

```
npm run serve
```

- 浏览器打开 http://localhost:8080/
